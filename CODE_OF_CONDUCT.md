# Code of Conduct

## Our Standards

This is a small, solo-maintained hobbyist project. Everyone who participates — opening issues,
submitting pull requests, or commenting — is expected to:

- Be respectful and constructive in all communication.
- Stay on topic (ADS-B, readsb, aviation tech, this project).
- Accept that the maintainer has the final say on scope and direction.

## Enforcement

Interactions that fall outside these standards may be hidden, edited, or removed at the
maintainer's discretion. Persistent problems can be reported privately via
[GitHub profile contact](https://github.com/blindp3w).

## Scope

This code of conduct applies to all project spaces: GitHub Issues, Pull Requests, Discussions,
and any other venue where the project is represented.
